Les évenements

Réaliser les exercices suivants en utilisant les évenements.

  1. Créer un bouton qui, lorsqu'il est cliqué, affiche une alerte avec le message "Bonjour !" (utiliser onclick).
  2. Créer un champ de texte qui, lorsqu'il est modifié, affiche dans la console le texte actuel du champ.
  3. Créer un bouton qui, lorsqu'il est survolé, change la couleur de fond de la page en rouge via une classe (lorsqu'il n'est plus survolé, il faut enlevé la couleur rouge).
  4. Créer un bouton qui, lorsqu'il est cliqué, cache un élément de la page.
  5. Créer un bouton qui, lorsqu'il est cliqué, affiche l'heure actuelle dans une alerte.
