Réaliser les exercices suivants en créant des try catch finally

  1. Créer une fonction qui prend un tableau de nombres et retourne la somme de ces nombres. Si le tableau est vide, lancer une exception avec le message "Le tableau est vide".
  2. Créer une fonction qui prend un nombre et retourne son inverse. Si le nombre est égal à zéro, lancer une exception avec le message "Division par zéro".
  3. Appeler les 2 fonctions et gérer l'exception en affichant le message d'erreur dans la console.
